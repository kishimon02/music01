# アーキテクチャノート

関連する詳細設計書: `docs/design.md`

## 現在のフェーズ

このリポジトリでは、Phase A の基盤のみを実装しています。

- 決定論的なミキサーグラフ対象（`MixerGraph`）
- 内蔵FXチェーンと固定パラメータID
- 将来のミキシング自動化に向けた分析スナップショットスキーマ
- 手動適用のみの提案コマンドフロー（apply/revert）
- MCPJ移行契約（`v1 -> v2`）

## ドメイン境界

- `music_create.mixing`
  - 実行時ミキシング状態と自動化に安全なコマンドモデル
- `music_create.project`
  - ファイルスキーマと移行ルール
- `music_create.api`
  - ミックス解析/提案の契約エンドポイント

## 今後の統合ポイント

- プレースホルダー解析をDSP品質の特徴抽出に置き換える
- pybind11経由でC++/JUCE音声コアと接続する
- 提案の比較・適用・巻き戻し履歴UIを追加する
- 内蔵FX中心から、制御可能なVST3機能へ拡張する
